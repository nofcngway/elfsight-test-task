# Rick and Morty Characters Explorer

Тестовое задание от **Elfsight**. Приложение для поиска и просмотра персонажей вселенной "Рик и Морти" с использованием официального API.

---

### Стек технологий

*   **Core:** [React](https://reactjs.org/)
*   **Styling:** [Styled Components](https://styled-components.com/)
*   **Networking:** [Axios](https://axios-http.com/)
*   **Quality & Style:** [ESLint](https://eslint.org/) + [Prettier](https://prettier.io/)

---

### Быстрый запуск

Для удобства используется **Makefile**. Выполните следующие команды в терминале:

1.  **Установка зависимостей:**
    ```bash
    make install
    ```
2.  **Запуск в режиме разработки:**
    ```bash
    make start
    ```

После запуска приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

---

### Сборка и Линтинг

*   **Production build:**
    ```bash

    ```
*   **Проверка кода:**

### Что реализовано

#### Техническая часть
*   **Исправлен баг:** Устранен бесконечный Loader в Popup.
*   **Рефакторинг:** Компонент `Card` разбит на логические подкомпоненты для лучшей читаемости и поддержки.
*   **Качество:** Код приведен в соответствие с правилами ESlint/Prettier.

#### Фильтрация и поиск
*   **Поля поиска:** Status, Gender, Species, Name, Type.
*   **Синхронизация:** Все примененные фильтры сохраняются в URL параметрах.
*   **Deeplinking:** При открытии страницы по ссылке фильтры автоматически применяются из URL.
*   **Управление:**
    *   `Apply` — применяет фильтры и возвращает на первую страницу.
    *   `Reset` — полностью очищает форму, сбрасывает URL и пагинацию.

#### UI/UX (Popup)
*   **Взаимодействие:** Закрытие по клику на подложку (overlay) или клавишей `Esc`.
*   **Комфорт:** Блокировка прокрутки (body scroll lock) при открытом модальном окне.
*   **Визуал:** Отображение гендерных иконок изначально работало корректно и не требовало исправлений.
